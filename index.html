Morning Report Alpha

매일 여러 소스에 흩어진 뉴스·시세를 직접 찾아보지 않아도, 시장 개장 전 자동으로 정리된 투자심리 리포트를 받아보는 개인용 자동화 서비스입니다.

라이브 대시보드 ↗ GitHub ↗

Problem

최근 주식 시장이 활황인 데 비해 정작 시장을 판단할 지식과 정보는 부족하다고 느꼈고, AI로 종목별 투자 심리를 분석해보고 싶다는 개인적인 동기에서 시작했습니다. 한국·미국 주요 20개 종목의 뉴스와 시세를 매일 여러 소스에서 따로 확인해야 하는 번거로움도 함께 해결하고자 했습니다.

Hypothesis

종목별 뉴스 기사 분석 점수와 실제 시장 가격의 움직임이 비슷하게 나타날 것이라고 가정했습니다.

MVP

  • 뉴스·가격 수집: Google News RSS와 Yahoo Finance API로 뉴스와 종가 데이터 수집
  • AI 분석: Gemini 2.5 Flash-Lite Structured Output으로 JSON 스키마 기반 결과 생성
  • 데이터 누적: CSV/JSON으로 이력 저장
  • 대시보드: Cloudflare 배포 정적 대시보드에서 종목별 조회 및 차트 제공
  • 알림: Telegram 리포트 자동 발송
  • 자동화: GitHub Actions로 한국장·미국장 개장 전 시간대에 맞춰 각 시장별로 뉴스 수집과 투자심리 리포트 생성을 개별 실행

My Role

기획, 수집·분석 스크립트 개발, 대시보드 구현, 배포·스케줄링 설정까지 기획부터 운영까지 전 과정을 혼자 맡아 진행한 개인 프로젝트입니다.

Key Decisions

서버와 DB를 새로 두지 않고, 무료로 유지 가능한 인프라만으로 전체 워크플로를 끝까지 운영 가능한 형태로 만드는 것을 우선했습니다.

  • Node.js로 개발한 수집·분석 스크립트를 GitHub Actions cron으로 지정된 시간에 자동 실행해 상시 서버를 두지 않음
  • 수집 데이터는 CSV에 append한 뒤 자동 commit·push해 별도 DB 없이 이력 관리
  • 대시보드는 Cloudflare로 배포하고 도메인 연결까지 포함해 무료 호스팅
  • 서버·DB 비용 없이 전체 서비스를 무료로 구축·운영

Current Result

현재 개인용으로 스케줄에 따라 자동 수집·분석·배포·알림까지 동작 중이며, 라이브 대시보드(investing.twojh.com)로 공개돼 있습니다. 뉴스 분석 점수와 실제 가격 움직임이 실제로 비슷하게 맞아떨어지는지에 대한 검증은 아직 진행 중입니다.

Learnings

무료 인프라만으로도 수집 → 분석 → 저장 → 배포 → 알림까지 이어지는 파이프라인을 처음부터 끝까지 직접 운영 가능한 형태로 만들 수 있다는 것을 확인했습니다. 또한 AI를 활용한 자동화에서는 분석 실패, 타임아웃, API rate limiting 같은 돌발 상황에 대한 세심한 예외 처리가 필요하다는 것도 배웠습니다.
