사용자가 2026-07-11 대화에서 명시적으로 "앞으로 답변은 한글로 해주면 좋겠어"라고 요청함.

**Why**: 이력서 프로젝트 소유자는 한국어 화자이고, 사이트 자체도 한국어 우선(Korean-first)으로 만들고 있음. 대화도 결과물과 같은 언어로 진행하는 것을 선호.

**How to apply**: 이 저장소(my-resume) 관련 대화에서는 코드/커밋 메시지 컨벤션 등 기술적으로 영어가 자연스러운 부분(변수명, 클래스명, 커밋 메시지 등)을 제외하고, 사용자에게 보내는 답변 텍스트는 한글로 작성할 것. [[feedback-analysis-before-edit-and-visible-todos]]와 함께 적용.
